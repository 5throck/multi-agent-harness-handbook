Workshop Pre-Installation Checklist

Complete this checklist before running the installation script. The items below cannot be automated and require manual action.

1. Verify Subscription (Top Priority)

The tools used in the workshop hands-on sessions require a valid subscription.

ToolRequired Plan
Claude Code CLI (claude)Claude Pro or Max
Claude Desktop AppClaude Pro or higher
Antigravity CLI (agy)Gemini Advanced
Antigravity Desktop AppGemini Advanced
Regarding API Keys: API keys are only needed for integrations outside of your subscription plan. The workshop hands-on sessions are fully covered by the subscription model, so only issue an API key if it is truly necessary.

2. Account Preparation

  • Google Account — required to use Antigravity / Gemini Advanced
  • GitHub Account — required for gh auth login and PR exercises
  • If you don't have one, sign up in advance at github.com/signup, following the sign-up guidance in Chapter 4 §2-A Practice Preparation Section

3. System Requirements

  • OS: Windows 11 (Intel) / macOS Sequoia 15 or later on Apple Silicon (M series) / Ubuntu 24.04 or later
  • Free disk space: 5 GB or more
  • Administrator / sudo privileges
  • Windows: Start → search "PowerShell" → right-click → "Run as administrator" (or Win + X → "Terminal (Admin)"), then click "Yes" in the UAC confirmation dialog
  • macOS / Linux: prefix the command with sudo
  • Stable internet connection (total download size approx. 1–2 GB)

4. Git User Setup

After installing git, register your user information with the commands below.

git config --global user.name  "Jane Doe"
git config --global user.email "you@example.com"
  • git config --global user.name configured
  • git config --global user.email configured

5. GitHub Authentication

gh auth login

Follow the prompts to complete authentication in your browser.

  • Confirm "Logged in" is displayed when running gh auth status

6. Project Environment Setup (after cloning the repository)

When using the co-develop or co-consult variants:

cp .env.sample .env
# Open the .env file and fill in the required values
  • Created .env from .env.sample
  • Entered required API keys / configuration values

co-develop Python projects — use uv (required):

uv sync          # Install dependencies based on pyproject.toml
# Or, if using requirements.txt:
uv pip install -r requirements.txt
  • Confirmed uv is installed and available (uv --version)
  • Installed Python dependencies via uv
Docker is optional. Install it only if the facilitator specifically instructs you to. Add the --docker flag on macOS/Linux or the -Docker flag on Windows to the installation script.
  • (Optional) Confirm Docker is installed and the daemon is running

7. Final Environment Check (must be run before the workshop)

Run the following command from the project root folder.

bun setup-common.ts

You are ready once every item in the output table shows ✅.

For questions, please contact the facilitator before the workshop begins.
