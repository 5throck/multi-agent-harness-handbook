기존 스타일 유지 + 다크모드 추가

기존 GitHub 스타일 CSS 변수는 그대로 유지합니다. 다크모드만 GitHub Dark 팔레트로 추가:

상단 우측의 ☀️ 버튼을 클릭하면 GitHub Dark 스타일의 다크모드가 전환됩니다. OS 시스템 설정의 다크모드도 자동 감지합니다.

언어 전환기 동작

상단 우측의 🌐 드롭다운에서 언어를 선택하면:

  • 01_Why_AI_Chapter.html → 한국어 (기존)
  • 01_Why_AI_Chapter_en.html → English (신규)
  • 01_Why_AI_Chapter_ja.html → 日本語 (신규)

선택된 언어는 localStorage에 저장되어 다른 페이지로 이동해도 유지됩니다.

파일명 규칙

모든 기존 파일은 그대로 유지되며, 접미사로 언어 변형이 추가됩니다:
